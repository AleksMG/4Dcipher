Криптостойкий S-Box с привязкой к ключу

Реализация криптосистемы с S-Box 16×16×16, где каждый входной байт заменяется на один из 16 возможных выходных байтов. S-Box детерминированно генерируется из ключа с использованием криптостойкого хеширования.

Алгоритм работы:

1. Ключ хешируется с помощью SHA-256 для получения начального состояния

2. На основе хеша строится детерминированный псевдослучайный S-Box 16×16×16

3. Каждый байт исходных данных заменяется случайным значением из соответствующей ячейки S-Box

4. Для дешифрования используется тот же S-Box, воссозданный из ключа

Ввод данных

Готов к работе. Введите ключ и данные.
