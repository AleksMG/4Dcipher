Визуализация куба 4×3

Результат:

Как работает шифр:

1. Символы алфавита размещаются в кубе 4×3

2. Каждый символ имеет координаты (строка, столбец)

3. Для шифрования: к координатам символа текста прибавляются координаты символа ключа (по модулю размеров куба)

4. Для расшифрования: из координат символа шифротекста вычитаются координаты символа ключа (по модулю размеров куба)
